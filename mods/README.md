# Mods

`mjolnir-xbox-styler.wh.cpp` is the canonical Windhawk mod source.

Workflow:
1. Edit/compile the mod inside the Windhawk app ("Create new mod" / mod editor).
2. Keep this file as the source of truth here in the repo.
3. To publish/update: copy it into a fork of `ramensoftware/windhawk-mods`
   under `mods/mjolnir-xbox-styler.wh.cpp` and open a PR.

The mod's `@github` metadata must be `yusufaf` (must match the PR author).
